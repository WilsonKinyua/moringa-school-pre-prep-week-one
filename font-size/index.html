Font-size

Besides pixel values, we can also set the font size using absolute size values relative to the default value. These values are:

  • xx-small
  • x-small
  • small
  • medium
  • large
  • x-large
  • xx-large

The default font size is medium.

Using the CSS font-weight property, we can make entire elements bold. However, at times we just want to make a few words in an element bold, for example in a paragraph, something important may be made bold. In this case, strong tags are used to show words that are important.

Emphasizing a few words in

Emphasis tags are used to emphasize words without using CSS styling. Most browsers will display the Em element as italicized text.
